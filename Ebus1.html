Multi Experience

Multi-experience development platform. A multi-experience development platform (MXDP) is a development platform that is used for developing not just mobile and web applications, but also chat, voice, augmented reality and wearable experiences

It helps tailor the technology to the user to become more user accessable, using UX design.

Distributed Cloud

A distributed file system for cloud is a file system that allows many clients to have access to data and supports operations on that data. Each data file may be partitioned into several parts called chunks. Each chunk may be stored on different remote machines, facilitating the parallel execution of applications.
